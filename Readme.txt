Restaurant Website

This repository contains the code for a fully responsive restaurant website.
The website is designed to provide a seamless user experience and includes pages for Home, Feed, Reserve a Table, Contact Us, and Menu. The website is built using HTML, CSS, JavaScript, and Bootstrap.
Features

1. Home Page
A welcoming section with a brief introduction to the restaurant.

2.About Us 
about us page provides information regarding the restaurant

3. Reserve a Table Page
Interactive form for customers to book a table.
provide audio guideness also

4. Contact Us Page
Provides contact information about Restaurant

5. Menu Page
Showcases the restaurant's dishes with  prices, and images.


Technologies Used

HTML: Structure of the website.

CSS: Styling and layout.

JavaScript: Interactive features and functionality.

Bootstrap: Responsive design and pre-built components.
Organized into categories like appetizers, main courses, desserts, and beverages.
